# Guía de Configuración del Proyecto

Sigue estos pasos en orden para levantar el repositorio correctamente.

## Pasos para levantar el proyecto

### 1. Prender el Backend
- Navega hasta la carpeta del backend (si no estás ya en ella).
- Ejecuta el comando correspondiente para iniciar el servidor

### 2. Inyectar el seed.js
- para poblar la base de datos con datos iniciales mediante el seed.js.

### 3. Instalar las dependencias en el Frontend
- Ubícate en la carpeta del frontend.
- instalar las dependencias

### 4. Ubicarse en la carpeta del Backend (con el comando cd)
- Asegúrate de estar en la raíz del proyecto o en la carpeta correspondiente.
-  posicionate correctamente si es necesario (con el cd).

### 5. Prender el Frontend
- Una vez instaladas las dependencias, inicia el servidor del frontend.
